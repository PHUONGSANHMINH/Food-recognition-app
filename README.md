# Food-recognition-app
A> Setup and run server API
---------------------------
1. git clone https://github.com/IL-Junq/Food-recognition-app.git
2. cd Food-recognition-app
3. cd Backend-API
4. python -m venv venv
5. venv\Scripts\activate
6. pip install -r requirements.txt
7. python app.py

B> Setup Frontend.
---------------------------
B.1> Admin
1. cd Frontend-Admin
2. npm install
3. npm start

B.2> Mobile
1. cd Frontend-Mobile
2. npx expo start
3. scan QR or open adb device
